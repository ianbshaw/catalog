DOCUMENTATION
-------------

THE ITEM CATALOG PROGRAM IS DESIGNED TO DISPLAY A MAIN MENU PAGE DISPLAYING ALL CATEGORY SUBJECTS.

REGISTERED USERS ARE ABLE TO ADD/EDIT/DELETE SUBJECTS AS WELL AS ITEMS.

USERS WILL BE ABLE TO REGISTER ON THE APP WITH GOOGLE CREDENTIALS.

INITIAL SETUP REQUIRES BUILDING THE DATABASE WITH python database_setup.py

THIS APP MAY BE RAN LOCALLY BY STARTING THE WEBSERVER WITH python project.py AND BROWSING TO http://localhost:5000
